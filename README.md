# Portfolio

Мой сайт-портфолио

## Структура файлов

```
portfolio/
├── index.html             # Главная страница (собранная из компонентов)
├── components/            # Переиспользуемые компоненты
│   ├── header.html        # Шапка сайта с навигацией
│   ├── hero.html          # Приветственная секция с типинг-эффектом
│   ├── tech-marquee.html  # Карусель технологий
│   ├── services.html      # Секция услуг
│   ├── projects.html      # Секция проектов
│   ├── contact.html       # Секция контактов
│   └── footer.html        # Подвал сайта
├── includes/              # Общие части
│   ├── head.html          # HTML head с мета-тегами и стилями
│   └── scripts.html       # JavaScript и внешние скрипты
├── 404.html               # Страница 404
├── manifest.webmanifest   # Web App Manifest
├── sitemap.xml            # XML карта сайта
└── robots.txt             # Файл для поисковых систем
```
